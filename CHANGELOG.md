# 변경 이력

## [0.3.0] - 2024-XX-XX (사용자 친화적 인터페이스)

### 추가
- Windows 배치 파일 (`setup.bat`, `run.bat`, `run_interactive.bat`)
  - 자동 설치 및 설정
  - 대화형 사용자 인터페이스
  - 명령줄 실행 지원
- 사용자 가이드 문서
  - `사용자_가이드.md`: 상세 사용 설명서
  - `README_사용자.md`: 빠른 시작 가이드
- Alt 텍스트 자동 생성 파이프라인 통합
- 접근성 검증 외부 도구(veraPDF) 연동 옵션
- 기본 StructTreeRoot 연결 로직 추가

### 개선
- 일반 사용자를 위한 간편한 실행 방법 제공
- 자동 환경 설정 및 의존성 설치
- 사용자 친화적인 오류 메시지

## [0.2.0] - 2024-XX-XX (고도화)

### 추가
- Alt 텍스트 자동 생성 모듈 (`src/tagger/alt_text_generator.py`)
  - GPT-4 Vision API 통합
  - WCAG 2.1 AA 기준 Alt 텍스트 생성
  - 주변 문맥 기반 설명 생성
  - 후처리 및 검증 기능

### 개선
- 태그 매칭 모듈에 Alt 텍스트 생성 기능 통합
- 문서 구조 분석 정확도 향상
- 에러 처리 및 로깅 강화

### 알려진 제한사항
- PyPDF의 구조 트리(StructTreeRoot) 생성 기능이 제한적임
- 완전한 PDF/UA 구조 트리 생성을 위해서는 추가 라이브러리 필요
- PAC 3/veraPDF 자동 검증은 아직 구현되지 않음

### 추가

- 태그 매칭 모듈
- PDF 생성 모듈
